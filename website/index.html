Sometimes when self-discipline is not enough, you just need a bit of help to stay on task.

With Eclipso's website blocker, you can make sure you stay locked in on work by inputting websites that you're not allowed to access during focus sessions. As long as the timer is running, the distracting tab will be closed for you and a nudging notification will be sent.

How the code works: when you start a focus session, Eclipso calculates and stores the time at which it will end as well as the list of blocked websites. Every time a tab is opened or changed, the code checks if its URL matches any of the blocked sites. If a match is found... DEATH BY DELETION.

Note that after you click "save", it will show "saaved!" for a few seconds before returning to the way it looked before. This doesn't mean the value has been lost :D it's all good!

Sleep is crucial to stay both physically and emotionally healthy. That's why it's really important to get a good night of sleep!

Time flies when you're having fun, or when you're focused. Or maybe you're doomscrolling and you need a reminder that it's getting late. Eclipso will notify you to get ready for bed at the time you set here!

How the code works: your input is saved in your browser's local storage and Eclipso calculates when that time occurs next to schedule a notification. The program listens for the alarm, when it'll show a notification, then sets another alarm for the next day. Aka, don't ruin a good night of sleep with a bad next one!

Again, note that after you click "save", it will show "saaved!" for a few seconds before returning to the way it looked before. The notification will still pop up at that time a-okay until you change it.
